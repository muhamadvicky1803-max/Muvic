# Kado Ulang Tahun Spesial 💖🎁

Website ini adalah kado ulang tahun digital spesial yang dirancang penuh cinta. Menampilkan UI interaktif yang romantis, animasi lucu, galeri kenangan bergaya lembaran buku, dan integrasi pemutar musik Spotify.

## ✨ Fitur Utama
- **Custom Password Lock**: Hanya bisa dibuka dengan PIN khusus (tanggal lahir).
- **Interactive Multi-Scene**: Alur cerita interaktif dengan berbagai GIF menggemaskan.
- **Buku Kenangan (Interactive Gallery)**: Galeri foto bergaya *scrapbook* yang halamannya bisa digeser (*swipe/slide*).
- **Smart Spotify Integration**: Menggunakan Spotify IFrame API. Jika lagu di Spotify diputar, lagu latar utama web otomatis berhenti.

## 🛠️ Cara Menjalankan Secara Lokal
1. Pastikan Anda memiliki browser modern.
2. *Clone* repository ini:
   ```bash
   git clone https://github.com/Oman21-skanic/kado-ulang-tahun-01.git
   ```
3. Buka file `index.html` langsung di browser, atau gunakan Live Server / `npx serve .`.

## 🌐 Cara Menyalakan GitHub Pages
Untuk membuat *website* ini *online* dan bisa diakses oleh si dia:
1. Buka halaman repository ini di GitHub.
2. Klik tab **Settings** (Pengaturan).
3. Di panel sebelah kiri, pilih **Pages**.
4. Di bagian *Build and deployment* -> *Source*, pilih **Deploy from a branch**.
5. Pada *Branch*, pilih **`main`** lalu klik **Save**.
6. Tunggu beberapa menit, lalu *refresh* halaman tersebut untuk mendapatkan link *website* *online* Anda!

Dibuat dengan ❤️ untuk momen yang tak terlupakan!
